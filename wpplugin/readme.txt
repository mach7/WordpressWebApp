Companion plugin for the WebApp starter. Adds roles/capabilities and integration helpers for the app.

== Description ==
This plugin provides minimal scaffolding (roles/caps and integration helpers) for a WordPress-integrated web application that lives alongside the WordPress install. API endpoints live under `webapp/api` in this template.

== Installation ==
Upload to `wp-content/plugins/webapp` and activate via the WordPress admin or WP-CLI.
